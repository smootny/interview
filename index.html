1. Jakie znasz nowości w Angularze 18+?

Odpowiedź: Angular 18+ wprowadził:

  • Standalone Components: Eliminacja konieczności deklaracji w modułach.
  • Signals: Nowy sposób zarządzania reaktywnością poprzez obserwowanie zmian danych.
  • Typed Forms: Formularze z obsługą typów w TypeScript.

Przykład Standalone Components:

import { Component } from '@angular/core';

@Component({
  standalone: true,
  selector: 'app-standalone',
  template: `
Standalone Component
`
})
export class StandaloneComponent {}

Przykład Signals:

import { signal } from '@angular/core';

// Tworzenie sygnału
const counter = signal(0);

// Subskrybowanie zmian
counter.subscribe(value => console.log('Nowa wartość:', value));

// Aktualizacja wartości
counter.set(counter() + 1); // Zwiększenie wartości o 1

2. Jakie 3 rzeczy lubisz w Angularze, a jakie 3 rzeczy nie lubisz w Angularze?

Lubię:

  • Modularność i lazy loading.
  • Ścisła integracja z TypeScript.
  • Bogaty ekosystem z wbudowanymi narzędziami.

Nie lubię:

  • Wysoka krzywa uczenia się.
  • Rozbudowany kod (verbosity).
  • Długie procesy budowania w dużych projektach.

3. Jakie widzisz znaczące różnice między Angular a Vue?

Odpowiedź:

  • Krzywa uczenia się: Angular jest trudniejszy niż Vue.
  • Reaktywność: Angular wymaga RxJS, Vue opiera się na automatycznej reaktywności.
  • Ekosystem: Angular to „framework all-in-one,” Vue jest lżejszy.

4. Jakich 3–5 bibliotek użyłbyś w nowym projekcie Angularowym?

Odpowiedź:

  • RxJS: Zarządzanie strumieniami danych.
  • NgRx: Zarządzanie stanem.
  • Angular Material: Profesjonalny UI.
  • ngx-translate: Wielojęzyczność.
  • Jest/Karma: Testowanie jednostkowe.

5. Czy wiesz, co to jest NgRx i RxJS?

Odpowiedź:

NgRx: Biblioteka zarządzania stanem, oparta na Redux.

// Akcja
export const increment = createAction('[Counter] Increment');

// Reducer
export const counterReducer = createReducer(0, on(increment, state => state + 1));

RxJS: Biblioteka do reaktywnego programowania opartego na Observable.

import { interval } from 'rxjs';

interval(1000).subscribe(val => console.log(val));
